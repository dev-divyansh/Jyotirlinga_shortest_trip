function tsp(start):
    visited = set()
    currentCity = start
    route = [currentCity]
    totalDistance = 0

    while visited.size < number_of_cities:
        visited.add(currentCity)
        nearestCity = null
        shortestDistance = Infinity

        for city in distances[currentCity]:
            if city not in visited and distances[currentCity][city] < shortestDistance:
                nearestCity = city
                shortestDistance = distances[currentCity][city]

        if nearestCity:
            route.append(nearestCity)
            totalDistance += shortestDistance
            currentCity = nearestCity
        else:
            break

    totalDistance += distances[currentCity][start]
    route.append(start)

    return { route, totalDistance }

This project leverages the Traveling Salesman Problem (TSP), a classic algorithmic problem in the field of computer science and operations research. By implementing a solution to the TSP, the project demonstrates the application of graph theory, optimization techniques, and algorithm design. The benefits of this project in an academic context include:

  • Practical Application: Applying theoretical concepts to solve real-world problems.
  • Algorithm Design: Understanding and implementing heuristic and exact algorithms.
  • Optimization: Enhancing problem-solving skills in minimizing costs and distances.
  • Data Structures: Utilizing appropriate data structures for efficient computation.
  • Interdisciplinary Learning: Combining knowledge from mathematics, computer science, and logistics.

Additionally, this project fosters critical thinking and analytical skills, which are essential for academic research and professional development.
